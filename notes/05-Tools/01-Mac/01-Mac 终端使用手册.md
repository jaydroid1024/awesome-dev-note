## 在“终端”窗口中工作

- 打开或退出“终端”

  - ```java
    xuejiewang@xuejiedeMacBook-Pro ~ % 
    //这表示名为 xuejiewang 的用户登录了名为“xuejiedeMacBook-Pro”的电脑，而波浪符号 (~) 表示当前文件夹为其个人文件夹。
    ```

  - ###### 退出 shell 会话

    - 键入 `exit`，然后按下 Enter/Return/回车 键。
    - 这会确保关闭 shell 中正在运行的命令。如果还有任何会话正在进行中，会出现对话框。

- 打开新的“终端”窗口和标签页

  - 描述文件是“终端”窗口的样式和行为设置的集合。

  - ###### 使用默认描述文件或与活跃窗口所使用相同的描述文件来打开新的“终端”窗口

    - 按下 Command-N

  - ###### 打开使用特定描述文件的新“终端”窗口

    - 选取“Shell”>“新建窗口”，然后从菜单末尾的描述文件列表中选取一个描述文件

  - ###### 使用默认描述文件或与活跃标签页所使用相同的描述文件打开新标签页

    - 按下 Command-T

  - ###### 打开使用特定描述文件的新标签页

    - 在 Mac 上的“终端” App 中，选取“Shell”>“新建标签页”，然后从菜单末尾的描述文件列表中选取一个描述文件。

- 执行命令和运行工具

  - ###### 在 shell 中执行命令

    - 如果命令位于 shell 的某个已知文件夹下，输入命令名称时省略路径信息。
    - 已知文件夹列表储存在 shell 的 PATH 环境变量中，而且包含大多数命令行工具的文件夹。
    - `ls` 展示当前用户个人文件夹中的文件
    - `open -a wechat.app`：打开一个APP
    - 输入命令时，如果收到 `command not found` 的信息，请检查您的拼写。

  - ###### 终止命令

    - 按下 `Control-C` 键会发出一个让大多数命令终止的信号。

  - ###### 重复之前输入的命令

    - 按下上箭头键，历史输入的命令就会显示在命令行上

- 指定文件和文件夹

  - 指向“终端” App 的路径为：`/System/Applications/Utilities/Terminal.app`
  - 表示特定文件夹的标准快捷键

  | 快捷键              | 描述                                                         |
  | ------------------- | ------------------------------------------------------------ |
  | 单个句点**.**       | 单个句点表示当前文件夹。例如，字符串“./Test.c”即表示当前文件夹中的 Test.c 文件。 |
  | 两个句点**..**      | 两个句点表示当前文件夹的父文件夹。例如，字符串“../Test”表示当前文件夹的同级文件夹（名为 Test）。 |
  | 波浪符号~[*用户名*] | 波浪符号表示已登录用户的个人文件夹。例如，若要指定当前用户的“文稿”文件夹，请输入“~/Documents”。若要指定另一位用户的“文稿”文件夹，请在短名称前加上波浪符号 (~)，例如“~username/Documents”。 |

- 纠正键入错误

  - *逐字符移动光标：*按下`左箭头键`或`右箭头键`。
  - *在光标位置插入字符：*键入字符。插入的位置是光标所在字符的前一个位置
  - *清除不想要的字符：*使用 `Delete 键`或按下 `Control-H` 键，然后键入新字符。
  - *清除行：*按下 `Control-U 键`。
  - *从光标位置清除到该行的结尾：*按下 `Control-K 键`。
  - *在当前命令中重新定位光标：*按住 `Option 键`，同时使用鼠标或触控板来重新定位光标。

- 将项目拖移到“终端”窗口中

  - 通过将项目拖移到“终端”窗口来输入文件的绝对路径或文本片段的内容，这样可以节省时间。
  - *输入文件或文件夹的路径：*将文件或文件夹拖移到“终端”窗口。项目的绝对路径将出现在命令行上。
  - *运行可执行文件：* 将文件拖移到“终端”窗口，然后按下 Return 键。
  - *使用存储为片段的文本摘录：* 将片段拖移到“终端”窗口。拖移片段可能会添加转义符，转义符等同于某些特殊文本字符。如果您不想包括这些字符，请在拖移片段时按住 Command 键。
  - *更改“终端”窗口的背景颜色：* 将颜色窗口或颜色池中的颜色拖移到“终端”窗口。

- 使用窗口组

  - 如果经常使用一组“终端”窗口，则可以将它们存储为一个窗口组，并在每次打开“终端”时自动打开它们。

  - 将窗口放置到想要的位置，然后为每个窗口配置所需的设置和标签页。这些设置会与组一起存储。

  - ###### 创建窗口组

    - 选取“窗口”>“将窗口存储为组”。
    - 输入窗口组的名称。
    - 如果您要在打开窗口组时自动恢复远程连接和无需确认即可安全运行的命令，请选择“恢复所有命令”。
    - 如果想要启动“终端”时打开这些窗口，请选择“当‘终端’启动时使用窗口组”。
    - 点按“存储”。

  - ###### 打开窗口组

    - 选取“窗口”>“打开窗口组”>“[*群组名称*]”。

  - ###### 移除窗口组

    - 选取“终端”>“偏好设置”。
    - 点按“窗口组”。
    - 选择窗口组，然后点按减号移除按钮

  - ###### 导入窗口组

    - 选取“终端”>“偏好设置”，然后点按“窗口组”。
    - 点按操作弹出式菜单，选取“导入”，选择包含该窗口组的文件，然后点按“打开”。
    - 将包含该窗口组的文件从“访达”拖移到窗口组列表。
    - 窗口组文件名以扩展名“.terminal”结尾。

  - ###### 导出窗口组

    - 选取“终端”>“偏好设置”，然后点按“窗口组”。
    - 选择您要导出的窗口组，点按操作弹出式菜单，选取“导出”，输入文件的名称，然后点按“存储”。
    - 将您要导出的窗口组从窗口组列表拖移到“访达”中。
    - 导出的窗口组文件名以扩展名“.terminal”结尾。



## 自定义终端

-  使用描述文件来更改“终端”窗口的外观

  - 描述文件是“终端”窗口的样式和行为设置的集合。包括窗口的颜色、字体、光标样式、背景和其他元素

  - “终端”附带一套预定义的描述文件，您也可以创建自己的自定描述文件。

  - ###### 创建新的描述文件或修改现有的描述文件

    - 选取“终端”>“偏好设置”，然后点按“描述文件”。
    - *创建新的描述文件*：点按描述文件列表下方的添加按钮，然后输入新描述文件的名称。
    - *修改现有的描述文件*：选择要修改的描述文件。

  - 为描述文件选取设置：

    - [文本](https://support.apple.com/zh-cn/guide/terminal/trmltxt/2.11/mac/11.0)：更改颜色、字体、文本、背景和光标样式。
    - [窗口](https://support.apple.com/zh-cn/guide/terminal/trmlwindw/2.11/mac/11.0)：更改窗口显示设置，包括标题、大小和回滚。
    - [标签页](https://support.apple.com/zh-cn/guide/terminal/trmltab/2.11/mac/11.0)：更改标签页显示设置，包括标题和进程名称。
    - [Shell](https://support.apple.com/zh-cn/guide/terminal/trmlshll/2.11/mac/11.0)：更改启动和退出行为的设置。
    - [键盘](https://support.apple.com/zh-cn/guide/terminal/trmlkbrd/2.11/mac/11.0)：更改功能键、Option 键和备用屏幕的设置。
    - [高级](https://support.apple.com/zh-cn/guide/terminal/trmladvn/2.11/mac/11.0)：更改终端类型 (Terminfo)、输入、报警声行为和多语言环境编码的设置。

    ![](https://tva1.sinaimg.cn/large/008eGmZEly1gnojbb2glnj30x10u0wuu.jpg)

  - ###### 恢复预定义的“终端”窗口描述文件

    - 如果移除或更改了“终端”附带的任何预定义描述文件，您可以将它们重新添加回描述文件列表，保留原始的设置。您仅可恢复整套预定义描述文件（也就是说，不可单独恢复任何预定义描述文件）。
    - 如果自定了其中一个预定义描述文件，并且想要存储自定内容，需要在恢复预定义描述文件前将其重新命名。恢复操作将会覆盖每个预定义描述文件的设置。
    - 选取“终端”>“偏好设置”，然后点按“描述文件”。
    - 点按操作弹出式菜单，然后选取“恢复默认描述文件”。
    - https://support.apple.com/zh-cn/guide/terminal/trml5856b1f2/mac
