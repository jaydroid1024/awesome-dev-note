## 在“终端”窗口中工作

- 打开或退出“终端”

  - ```java
    xuejiewang@xuejiedeMacBook-Pro ~ % 
    //这表示名为 xuejiewang 的用户登录了名为“xuejiedeMacBook-Pro”的电脑，而波浪符号 (~) 表示当前文件夹为其个人文件夹。
    ```

  - ###### 退出 shell 会话

    - 键入 `exit`，然后按下 Enter/Return/回车 键。
    - 这会确保关闭 shell 中正在运行的命令。如果还有任何会话正在进行中，会出现对话框。

- 打开新的“终端”窗口和标签页

  - 描述文件：描述终端窗口样式的文件，例如：字号、字色、背景色等

  - ###### 使用默认描述文件或与活跃窗口所使用相同的描述文件来打开新的“终端”窗口

    - 按下 Command-N

  - ###### 打开使用特定描述文件的新“终端”窗口

    - 选取“Shell”>“新建窗口”，然后从菜单末尾的描述文件列表中选取一个描述文件

  - ###### 使用默认描述文件或与活跃标签页所使用相同的描述文件打开新标签页

    - 按下 Command-T

  - ###### 打开使用特定描述文件的新标签页

    - 在 Mac 上的“终端” App 中，选取“Shell”>“新建标签页”，然后从菜单末尾的描述文件列表中选取一个描述文件。

- 执行命令和运行工具

  - ###### 在 shell 中执行命令

    - 如果命令位于 shell 的某个已知文件夹下，输入命令名称时省略路径信息。
    - 已知文件夹列表储存在 shell 的 PATH 环境变量中，而且包含大多数命令行工具的文件夹。
    - `ls` 展示当前用户个人文件夹中的文件
    - `open -a wechat.app`：打开一个APP
    - 输入命令时，如果收到 `command not found` 的信息，请检查您的拼写。

  - ###### 终止命令

    - 按下 `Control-C` 键会发出一个让大多数命令终止的信号。

  - ###### 重复之前输入的命令

    - 按下上箭头键，历史输入的命令就会显示在命令行上

- 指定文件和文件夹

  - 指向“终端” App 的路径为：`/System/Applications/Utilities/Terminal.app`
  - 表示特定文件夹的标准快捷键

  | 快捷键              | 描述                                                         |
  | ------------------- | ------------------------------------------------------------ |
  | 单个句点**.**       | 单个句点表示当前文件夹。例如，字符串“./Test.c”即表示当前文件夹中的 Test.c 文件。 |
  | 两个句点**..**      | 两个句点表示当前文件夹的父文件夹。例如，字符串“../Test”表示当前文件夹的同级文件夹（名为 Test）。 |
  | 波浪符号~[*用户名*] | 波浪符号表示已登录用户的个人文件夹。例如，若要指定当前用户的“文稿”文件夹，请输入“~/Documents”。若要指定另一位用户的“文稿”文件夹，请在短名称前加上波浪符号 (~)，例如“~username/Documents”。 |

- 纠正键入错误

  - *逐字符移动光标：*按下`左箭头键`或`右箭头键`。
  - *在光标位置插入字符：*键入字符。插入的位置是光标所在字符的前一个位置
  - *清除不想要的字符：*使用 `Delete 键`或按下 `Control-H` 键，然后键入新字符。
  - *清除行：*按下 `Control-U 键`。
  - *从光标位置清除到该行的结尾：*按下 `Control-K 键`。
  - *在当前命令中重新定位光标：*按住 `Option 键`，同时使用鼠标或触控板来重新定位光标。

- 将项目拖移到“终端”窗口中

  - 通过将项目拖移到“终端”窗口来输入文件的绝对路径或文本片段的内容，这样可以节省时间。
  - *输入文件或文件夹的路径：*将文件或文件夹拖移到“终端”窗口。项目的绝对路径将出现在命令行上。
  - *运行可执行文件：* 将文件拖移到“终端”窗口，然后按下 Return 键。
  - *使用存储为片段的文本摘录：* 将片段拖移到“终端”窗口。拖移片段可能会添加转义符，转义符等同于某些特殊文本字符。如果您不想包括这些字符，请在拖移片段时按住 Command 键。
  - *更改“终端”窗口的背景颜色：* 将颜色窗口或颜色池中的颜色拖移到“终端”窗口。

- 使用窗口组

  - 如果经常使用一组“终端”窗口，则可以将它们存储为一个窗口组，并在每次打开“终端”时自动打开它们。
